# Math

A really simple terminal program that allows users to perform some simple arithmatic in the console.

## Installation

```
git clone https://github.com/RizaHKhan/mathcli.git
cd mathcli
chmod +x mathcli
go build ./math.go
cp ./math ~/.local/bin/math

```

## Usage examples

```
math "1 + 1"

```

Expected results:

```
math "1 + 1"
2

```
